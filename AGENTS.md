# AI エージェント基本方針

## 目的
エージェント共通の作業方針を定義する。

## 基本方針
- 会話言語: 日本語
- コメント言語: 日本語
- エラーメッセージ言語: 英語
- コミット規約: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) に従う。`<description>` は日本語で記載する。

## 判断記録のルール
- 判断内容、代替案、採用理由、前提条件、不確実性を明示する。

## 開発手順（概要）
1. プロジェクト理解（`pom.xml`, `README.md`, ソースコードの確認）
2. 依存関係の解決（`mvn compile`）
3. 変更実装（既存のコードスタイルを尊重）
4. ビルドと動作確認（`mvn package`, JAR 実行確認）

## セキュリティ / 機密情報
- 認証情報のコミット禁止。
- ログへの機密情報出力禁止。

## リポジトリ固有
- このプロジェクトは systemd のログを追跡するため、`journalctl` コマンドが利用可能な Linux 環境を前提とする。
- Java 16 以上が必要。
